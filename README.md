# System Design Roadmap (Beginner → Advanced)

> A complete roadmap to learn System Design from scratch with logical
> thinking, coding practice, game-system design, and real-world
> projects.

## Topics

-   Computer Architecture
-   CPU
-   RAM

-   Threads
-   Stack & Heap
-   Memory Management
-   Virtual Memory
-   File Systems
-   Operating System Basics
-   Scheduling
-   Context Switching
-   Binary & Hexadecimal

## Objectives

-   Understand how computers execute programs.
-   Learn memory usage and performance.
-   Understand process and thread behavior.

## Practice

-   Explain why RAM is faster than SSD.
-   Explain the difference between a process and a thread.
-   Explain why browsers create multiple processes.

# Learning Strategy

For every topic:

1.  Learn the theory.
2.  Draw diagrams.
3.  Build a mini-project.
4.  Explain trade-offs.
5.  Scale the project.
6.  Document your design.
7.  Review and refactor.

Following this roadmap develops strong logical thinking and practical
system design skills suitable for production software, browser games,
SaaS platforms, and technical interviews.
